Quick 3-Minute Setup

üìå What is this for?
You need a Vercel API token to connect Claude Code to Vercel, allowing you to deploy and manage your projects directly from chat!

1 Go to Vercel Account Settings

Click the button below to open Vercel's token page:

üîó Open Vercel Tokens Page
‚ö†Ô∏è Make sure you're logged in to Vercel first!
If not logged in, go to https://vercel.com/login

2 Create New Token

üì∏ You'll see a page titled "Tokens" with a button "Create Token" or "+ Create"
  1. Click the "Create Token" button (or "+ Create")
  2. In the popup/form:
    • Token Name: Type Claude Code MCP
    • Scope: Select "Full Account" (or leave default)
    • Expiration: Choose "No Expiration" (or 1 year)
  3. Click "Create Token" button

3 Copy Your Token

üö® IMPORTANT: The token will only be shown ONCE! Copy it immediately and save it somewhere safe.
üì∏ A long string will appear (starts with "vercel_" or similar)
  1. Click the "Copy" button next to the token
  2. The token will look something like this:
Example token format:
vercel_api_ABCdef123XYZ789_1234567890abcdefghij
‚úÖ Token Copied? Great! Keep it safe - you'll need it in the next step.

4 Add Token to Claude Code

Now that you have your token, paste it into Termux and run this command:

claude mcp add vercel npx -e VERCEL_API_TOKEN=YOUR_TOKEN_HERE -- -y @vercel/sdk mcp start
üìù Replace YOUR_TOKEN_HERE with the actual token you copied!

Example:
claude mcp add vercel npx -e VERCEL_API_TOKEN=vercel_api_ABCdef123XYZ789_1234567890abcdefghij -- -y @vercel/sdk mcp start

What this command does:

  • claude mcp add vercel - Adds a new MCP server named "vercel"
  • npx - Uses npx to run the package
  • -e VERCEL_API_TOKEN=... - Sets your API token as environment variable
  • -- -y @vercel/sdk mcp start - Runs the Vercel SDK MCP server

5 Verify Connection

After adding the MCP server, verify it's connected:

claude mcp list

You should see vercel in the list with status "Connected" ‚úÖ

üéâ Success! You can now use Vercel commands from Claude Code!

Try asking:
  • "Deploy my app to Vercel"
  • "List my Vercel deployments"
  • "Add environment variables to Vercel"

üîß Troubleshooting

‚ùå "Command failed" or "Not found"

Make sure you're using the exact command format with -- separator before the npx arguments.

‚ùå "Invalid token"

Double-check that you copied the entire token correctly, including any prefix like vercel_.

‚ùå MCP server shows as "Disconnected"

Try removing and re-adding:

claude mcp remove vercel -s local claude mcp add vercel npx -e VERCEL_API_TOKEN=your_token -- -y @vercel/sdk mcp start

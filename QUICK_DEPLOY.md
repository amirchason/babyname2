# Quick Deploy Reference - SoulSeed

## FASTEST Deploy (10-30 seconds)
```bash
npm run deploy
```
**OR**
```bash
npm run ship
```

## Test Before Deploy
```bash
npm run deploy:preview
```

## One-Command: Commit + Deploy
```bash
npm run ship-msg
```

## Live URLs
- **Production**: https://soulseedbaby.com
- **Preview**: Check terminal output after `deploy:preview`

## Rollback
Go to: https://vercel.com/ss-9666de73/babyname2
Click: Deployments → Pick old version → Promote to Production

---
**Platform**: Vercel (10-30 sec deploys)
